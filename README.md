# GEREJA TORAJA JEMAAT SURABAYA

Website resmi Gereja Toraja Jemaat Surabaya untuk menyampaikan informasi, warta jemaat, dan pelayanan gereja kepada jemaat dan masyarakat.

## Teknologi yang Digunakan

- **Frontend**: Next.js
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **CMS**: Strapi (akan diintegrasikan)
- **Database**: Supabase (akan diintegrasikan)
- **Media Storage**: Cloudinary (akan diintegrasikan)

## Fitur Utama

- Landing page dengan informasi dasar gereja
- Sistem menu navigasi dengan sidebar collapsible
- Halaman warta jemaat dengan berbagai submenu:
  - Berita Gereja
  - Agenda Ibadah
  - Pengumuman
  - Renungan Harian
  - Laporan Keuangan
- Halaman kontak dengan form interaktif
- Sistem komponen UI yang dapat digunakan kembali

## Instalasi Lokal

1. Clone repository ini
2. Jalankan `npm install` untuk menginstal dependensi
3. Jalankan `npm run dev` untuk memulai server pengembangan
4. Buka `http://localhost:3000` di browser Anda
